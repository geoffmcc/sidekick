# Contributing to Sidekick

Contributions to Sidekick are welcome through GitHub pull requests.

By submitting a contribution to this repository, you agree that your contribution is licensed under the GNU General Public License version 3 only (`GPL-3.0-only`).

You must have the right to submit your contribution. Do not include code, assets, credentials, private information, or other material that you are not authorized to contribute.

Keep changes focused, document notable behavior changes, and include appropriate tests where applicable.
